# INSTALLATION

## Prerequisites

1. [Node](https://nodejs.org/en/download/) version 16
2. [Yarn](https://yarnpkg.com/getting-started) verion >=1.22.19
3. [Expo CLI](https://docs.expo.dev/workflow/expo-cli/)
4. [Git](https://git-scm.com) SCM
